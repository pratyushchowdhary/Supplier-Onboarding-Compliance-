# Supply NOW Supplier Portal

Supplier-side onboarding MVP for the Supply NOW supplier, compliance, and procurement platform.

## Run locally

```bash
npm install
npm run dev
```

## Current scope

- Public supplier onboarding entry page
- Supplier application dashboard
- Document requirement tracking
- Business information form
- Supply item form
- Review and submission readiness
- Shared design tokens and reusable UI components
- Local draft persistence

The current release is a frontend MVP. Authentication, database persistence, object storage, document processing, and compliance workflow APIs are the next backend phase.
